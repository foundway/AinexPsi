Docker image for ROS2 Humble Perception generated by the following command:
'''docker pull ros:humble-perception'''

Bring up the docker image as a container:
'''docker compose up -d'''

Enter the shell in the container:
'''docker exec -it ros-humble-container bash'''

Put down the container:
'''docker compose down'''
